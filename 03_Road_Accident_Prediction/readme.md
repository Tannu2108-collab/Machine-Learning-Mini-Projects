# Road Accident Prediction

A Machine Learning project to predict the severity of road accidents based on traffic and location data.

## 🛠️ Tech Stack
* **Python**
* **Pandas & NumPy**
* **Scikit-Learn** (Classification algorithms)
* **Matplotlib/Seaborn** (Visualization)

## 📂 Project Overview
This project classifies accident severity levels to help identify high-risk zones and improve road safety measures.
