# Simple House Price Prediction

A beginner-friendly machine learning project that predicts house prices based on area using **Simple Linear Regression**.

## 🛠️ Tech Stack
* **Python**
* **Pandas** (Data manipulation)
* **Scikit-Learn** (Linear Regression model)
* **Matplotlib** (Data visualization)

## 📂 Project Overview
This project demonstrates the fundamental concept of mapping a single feature (Area) to a target value (Price).

## 🚀 How to Run
1. Ensure you have the required libraries installed:
   `pip install pandas scikit-learn matplotlib`
2. Run the script:
   `python house_price.py`
