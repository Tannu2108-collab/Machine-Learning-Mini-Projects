# Advanced House Price Prediction

A professional-grade machine learning project utilizing **Multiple Linear Regression** to predict house values using the **California Housing Dataset**.

## 🛠️ Tech Stack
* **Python**
* **Pandas** & **NumPy**
* **Scikit-Learn** (Preprocessing, Modeling, and Evaluation)
* **Matplotlib**

## 📂 Key Features
* **Multiple Features:** Predicts prices based on various housing metrics.
* **Model Evaluation:** Includes **Mean Squared Error (MSE)** and **R2 Score** to measure accuracy.
* **Performance Metrics:** Demonstrates professional model training and testing split.

## 📊 Evaluation Metrics
* **R2 Score:** Measures the proportion of variance in the dependent variable.
* **MSE:** Analyzes the average squared difference between estimated and actual values.
